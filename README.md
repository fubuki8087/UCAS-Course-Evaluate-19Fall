# UCAS-Course-Evaluate-20Spring

课程自动评估，2020春季学期使用，学期更换会导致评估页面`EvaPage`地址改变。

#### 使用方法

- 在参数`onestop_data`与中填入SEP用户名及密码。

- 在参数`cdata`与`tdata`中修改课程与老师评语（若需要）。

- 运行程序，等待完成。
